# Standard Article Template

Use this template for essays, reflections, project notes, and public updates.
Copy the markdown block into a new file under `articles/`, replace every
bracketed prompt, and remove sections that do not fit the post.

```markdown
---
title: "[Clear post title]"
description: "[One direct sentence describing the post]"
permalink: /articles/[post-slug]/
---

# [Clear post title]

[Opening paragraph that says what this post is about, why it exists, and what
the reader should expect. Keep this direct and specific.]

## Context

[Brief background, project setting, or motivation. Include only the context a
reader needs to understand the post.]

## Notes

[Main point one.]

[Main point two.]

[Main point three.]

## What I Learned

[Short reflection on the practical takeaway. Separate what happened from what
you infer from it.]

## Links

- [Related resource](https://example.com/)
```

## Review Before Publishing

- Replace every bracketed prompt.
- Remove `## Links` if the post does not depend on external resources.
- Check that claims are supported by the post itself or linked sources.
- Mark provisional, unfinished, or AI-assisted work clearly.
- Confirm the article is added to `index.md` and `articles/index.md` only when
  it should be public and discoverable.
